Welcome to PySpigot's Documentation!
====================================

PySpigot is a Bukkit/Spigot plugin designed to load and run Python scripts that can utilize the Bukkit/Spigot API.

PySpigot is compatible with Minecraft versions 1.8 through 1.16.5. Versions older than 1.8 may be supported, but no support is guaranteed for these versions.

* JavaDocs for the project can be viewed `here <https://docs.magicmq.dev/pyspigot/>`__.
* Check out PySpigot on GitHub `here <https://github.com/magicmq/PySpigot>`__.

For detailed documentation and information on writing scripts, see :ref:`here <writingscripts>`.

PySpigot API
############
